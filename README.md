Inicialmente, tentou-se utilizar as bibliotecas MediaPipe e DeepFace em conjunto, considerando que o DeepFace apresenta melhores resultados na detecção de emoções e facilidade de uso.

Entretanto, devido a incompatibilidades entre as bibliotecas, optou-se por utilizar exclusivamente o MediaPipe, na versão 0.10.20, em conjunto com o Python 3.12.12, sendo essa configuração necessária no ambiente de execução para o correto funcionamento do notebook.

Os resultados obtidos para a detecção de movimentos foram satisfatórios; contudo, no que se refere à identificação de emoções, a utilização do DeepFace mostrou-se superior, permanecendo sua integração como um desafio para trabalhos futuros.

Também fica como desafio para o futuro utilizar a biblioteca yolo.
